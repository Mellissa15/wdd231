Site Plan – Local Art Events Hub

Site Name

Local Art Events Hub

This name communicates the purpose of the site: to be a central hub for finding upcoming art events in Cape Town. It’s short, descriptive, and SEO-friendly.

Optional domain: art-eventshub.org

Site Purpose

The website showcases upcoming local art events, such as exhibitions, workshops, and gallery shows. It will include a dynamic events directory powered by JavaScript that fetches data from a JSON file or API. Users will be able to browse events, filter them by category or date, and submit inquiries through a contact form.

Scenarios

  • Which art exhibitions are happening this weekend?
  • How can I register for an upcoming painting workshop?

Color Scheme

  • Deep Indigo (#3f51b5): for navigation, headers, and call-to-action buttons.
  • Warm Coral (#ff7043): for highlights, active states, and icons.

Typography

Headings: Lora, serif
Body Text: Roboto, sans-serif

Wireframes

Home Page Layout Sketches

The wireframes show the hero section at the top, followed by a dynamic events list, featured galleries, and a footer with social links.
